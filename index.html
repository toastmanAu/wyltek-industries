A modular, board-agnostic embedded SDK for ESP32. Zero external dependencies ‚Äî 39+ sensor drivers, 32 board targets, camera support, animated UI modules. Every driver shares the same WySensorData interface: consistent, composable, and flash-efficient via conditional compilation.

ESP32 Arduino C++ Active

39+ sensor drivers ‚Äî BME280, MLX90640, LD2410, DS18B20, INA219, Wind, UV, and more
32 board targets ‚Äî CYD, Guition 4848S040, LilyGo T-Display/T-Deck/T-Watch, Waveshare, M5Stack, ESP32-CAM
WyCamera ‚Äî MJPEG stream server + motion detection for ESP32-CAM
WyEyes ‚Äî animated dual round-display robot eyes (GC9A01)
WySensors registry ‚Äî uniform addI2C / addSPI / addGPIO / addUART API
Conditional compilation throughout ‚Äî unused drivers cost zero flash
