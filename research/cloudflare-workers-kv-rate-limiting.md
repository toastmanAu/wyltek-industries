## Relevant Code/API Snippets
The `wrangler` CLI can initialize a basic Worker project:
```bash
npx wrangler init my-worker -y
```

An example `wrangler.jsonc` configuration, as generated by `npx wrangler init`, is structured as follows:
```jsonc
{
  "$schema": "node_modules/wrangler/config-schema.json",
  "name": "my-worker",
  "main": "./src/index.ts",
  "compatibility_date": "YYYY-MM-DD",
}
```
