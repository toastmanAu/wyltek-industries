# Quantum-Resistant Chains: Comparison of Infrastructure and Development

CoinMarketCap currently lists 17 **“Quantum-Resistant”** blockchain projects (e.g. Zcash, Starknet, Nervos, etc.)【1†L22-L30】【92†L339-L347】.  These include well-known L1s (Zcash, Nervos, QRL, Minima, Neptune), layer-2s (Starknet), and niche chains (Naoris, QANplatform, Abelian, etc.).  We examined each project’s official website and GitHub repositories to assess **infrastructure completeness** (node software, wallets, explorers, SDKs) and **development activity** (commits over the last 7 years, recency and gaps).  Based on this we devised a rubric with scores for *Activity* and *Infrastructure* (see **Methodology** below) and ranked the projects accordingly.  

**Key findings:** Some projects (e.g. **Minima, QRL, Neptune Cash, Nervos CKB, QuantumCoin**) have full stacks (open-source node, wallets, explorers, SDKs) and active development into 2025–2026【63†L203-L205】【86†L203-L211】. Others are largely one-off or marketing-driven: e.g. **QANplatform** and **Naoris** have little public code (mostly NDA/proprietary), **AME Chain** and **QuStream** show minimal recent commits, and **qONE** is essentially just a new token/toolkit on another chain【30†L269-L272】【59†L234-L237】【81†L247-L254】. **Methodology:** We scored each project (0–10) by combining:
- **Development Activity:** number and recency of commits on the main chain repo (we inspected GitHub commit histories).  Projects with active commits in 2025–2026 scored high (e.g. *Minima, Neptune, QRL*, see【63†L203-L205】【79†L421-L424】), whereas stalled projects scored low (e.g. *QAN, Naoris*【30†L269-L272】【81†L247-L254】).  We also noted long gaps or abandoned repos.
- **Infrastructure Completeness:** whether there is open-source **node software**, one or more **wallets/explorers**, and **developer tools/SDKs**.  A project with a full stack (node + multiple wallets + explorer + libraries) earned more points. For example, *Minima* has a Java full node and Android wallet (now archived)【62†L207-L212】【61†L206-L210】, a block explorer and JS SDK【61†L218-L222】. *Neptune* has a Rust node (`neptune-core`), GUI wallet (`neptune-proton`), explorer, and cryptographic libraries【84†L198-L207】【85†L213-L220】. By contrast, *QANplatform* and *Naoris* have virtually no public core code, so their infrastructure is incomplete【30†L269-L272】【34†L203-L211】.
- **Open Source / Transparency:** We favored projects that open their code (even if some IP is proprietary).  All our top scorers have their code on GitHub. Projects that are closed or require NDAs scored lower.
- **Project Maturity:** We roughly considered project age and mainnet status.  Mature chains like *Zcash* or *Nervos* automatically had the prerequisite of full infrastructure, but we still penalized lack of PQ content if out of scope (Zcash uses ZK-SNARKs, not PQ algorithms). 

We then combined these factors into an overall score.  For example, *Neptune Cash* and *Minima* both scored **~9.0–9.5/10** for having very active open development and complete feature sets【86†L203-L211】【63†L203-L205】. *QRL* and *Nervos* scored around **8.5** due to robust infrastructure and recent releases【79†L421-L424】【94†L207-L212】. At the bottom, *qONE* is barely a token/tool and got **1/10**【81†L247-L254】, while *QANplatform*, *Naoris*, and *AME Chain* are mostly incomplete and scored below **3/10**【30†L269-L272】【77†L258-L263】.

In summary, the **more mature, well-funded projects with long histories (Minima, QRL, Nervos, Zcash, Neptune, QuantumCoin)** generally form the “upper tier”: full implementations and ongoing development.  The “one-trick” or early-stage projects (*QANplatform, Naoris, AME, QuStream, qONE*) have little to show in open-source infrastructure or recent commits, despite marketing claims.  Nervos CKB stands out as fully in the league of these active chains: it has a working mainnet, full node and SDKs, and its repo shows merges as recently as Mar 2026【94†L207-L212】, on par with other top projects.
