FiberQuest

Retro gaming with real-money micropayments on CKB Fiber Network. Every game event — banana collected, life lost, score increased — triggers an instant payment between players.

⚡ Fiber Network 🎮 RetroArch 💰 Micropayments 🔓 Open Source 🤝 Community
⚙️ How It Works
1
RetroArch polls RAM The RAM Viewer reads game memory every 200ms via RetroArch's UDP network command interface.
2
Sidecar detects events A Node.js sidecar daemon watches for meaningful changes — banana collected, life lost, level complete — using verified RAM address maps.
3
Fiber channel payment Each event triggers an instant micropayment over a CKB Fiber Network channel. Payments settle in milliseconds, no on-chain fee per transaction.
4
Channel settlement At game end, the channel closes and the net result settles on-chain. Winner takes the agreed stake.
