Embed in your own page

One script tag, one div, one callback. Works anywhere — DOB minter, node dashboard, wallet tool.

<div id="my-node-status"></div>
<script src="https://wyltekindustries.com/js/ckb-node-finder.js"></script>
<script>
  new CKBNodeWidget('#my-node-status')
    .start()
    .on('connected', ({ url, tipBlock }) => {
      // url is ready — make RPC calls from here
      myApp.init(url);
    });
</script>

Note Chrome may show a permission prompt for localhost access (Local Network Access policy). The widget handles this — if blocked, it falls back to a manual URL input.

How it works

  1. Checks localStorage for a previously saved URL
  2. Tries localhost:8114 → :8117 → :9000 (3s timeout each)
  3. If nothing responds, shows a URL input prompt
  4. Validates by calling get_tip_block_number on your node
  5. Saves the working URL for next time

Privacy All traffic goes directly browser → your node. Nothing touches our servers.  View source ↗
