⚙️ Resources

Open-source tools, scripts, and guides built by Wyltek. Free to use — MIT licensed unless noted. Hosted on GitHub.

OpenClaw & AI Agents
⚡ openclaw-tg-webhook Tool
Switch OpenClaw's Telegram integration from long-polling to webhook mode. ~31x faster message delivery via Cloudflare Tunnel. One command setup, no router port forwarding needed.
OpenClaw Telegram Cloudflare Pi

CKB & Nervos Network
📸 ckb-node-dashboard Tool
Lightweight Node.js proxy + HTML dashboard for monitoring a CKB full node. Shows block height, peers, sync status. Polling-based, no dependencies.
CKB Node.js Dashboard
GitHub ↗
🐋 ckb-whale-bot Bot
Telegram bot that monitors CKB transactions and alerts when large transfers occur. Configurable USD threshold, live CKB price from CoinGecko, filters self-transfers.
CKB Telegram Node.js
GitHub ↗
🛡️ ckb-antiscam Bot
Anti-scam Telegram bot for CKB community groups. Detects trading spam, bot behaviour, and suspicious join patterns. Learn mode with precision/recall calibration before going live.
CKB Telegram Moderation
GitHub ↗
⛏️ ckb-stratum-proxy Tool
Stratum proxy for CKB mining. Handles ViaBTC protocol quirks, per-miner extranonce allocation, and provides stats via HTTP. Point your miners here instead of directly at the pool.
CKB Mining Stratum
GitHub ↗
🔥 spore-burner Tool
Browser tool to burn (melt) Spore/DOB NFTs on CKB. Reclaims locked CKB capacity. JoyID wallet support, testnet/mainnet aware, shows on-chain images inline.
CKB Spore DOB JoyID
Live ↗ GitHub ↗
Hardware & Embedded
⛏️ NerdMiner_CKB Tool
ESP32 Eaglesong miner for CKB. Based on NerdMiner, adapted for CKB's Eaglesong PoW. Targets the CYD (ESP32-2432S028R) with touchscreen display. PlatformIO build.
