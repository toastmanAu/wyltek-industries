🚀 OpenClaw Agent Setup

Hardware Requirements

  • Raspberry Pi 5 (8GB+ RAM) or equivalent x86
  • 8GB+ storage for agent + models
  • Stable internet connection

Installation (Pi 5)

# Update system
sudo apt update && sudo apt upgrade -y

# Install Node.js 22+
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

# Install OpenClaw
npm install -g openclaw

# Initialize agent
openclaw init --non-interactive
openclaw pairing approve telegram CODE_HERE

# Start
openclaw agent run

Configuration

Edit ~/.openclaw/openclaw.json:

{
  "agent": {
    "name": "Kernel",
    "model": "ckbdev/claude-sonnet-4-6"
  },
  "channels": {
    "telegram": { "enabled": true }
  },
  "plugins": {
    "entries": {
      "telegram": { "enabled": true }
    }
  }
}

# Install Ollama

Troubleshooting

  • Telegram auth fails: Ensure DM the bot first, then use correct code
  • Model timeouts: Increase timeout in config, check local inference health
  • Disk full: Models can be large; use quantized versions (GGUF)
